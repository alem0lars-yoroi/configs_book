# Semantic tools

## Protege

### Installation

```ShellSession
# emerge "dev-util/protege"
```

### Plugins

Install the following plugins (as root):

- Annotation Search Views.
- Annotation Template View.
- Beanshell View.
- Bookmark View.
- Cardinality View.
- Change View.
- Cloud Views.
- FaCT++ reasoner.
- HermiT Reasoner.
- DL-Learner.
- OWLDoc.

- Existential Query.
- Matrix Views.
