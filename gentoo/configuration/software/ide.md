# IDE

## Eclipse

### Installation

```ShellSession
# emerge "dev-util/eclipse-sdk-bin"
```

### Configuration

- Start `Eclipse`.
- Select the workspace (mine is:
  `/archive/development/workspaces/TestBench for Systemwide Eclipse`)
  and mark it as default workspace.
