# `ImageOptim`

## Requirements

* [Package Manager](../system/package_manager.md)

## Installation

Install tools used by `ImageOptim`:

```ShellSession
$ brew install pngcrush
$ brew install optipng
$ brew install jpegoptim
$ brew install zopfli
$ brew install advancecomp
$ brew install gifsicle
$ brew install pngout
```
