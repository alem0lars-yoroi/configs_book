# ZSH

## Package manager

```ShellSession
$ brew install antigen
```

## Configuration

The configuration should be available through the *VCSH configuration for ZSH* (the [configs-zsh](github.com/alem0lars/configs-zsh) repository contents)

* Set ZSH as the default shell:

  ```ShellSession
  $ chsh -s /bin/zsh
  ```
