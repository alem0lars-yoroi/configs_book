# Sublime Text

## Installation

```ShellSession
$ brew cask install sublime-text3 --appdir=/Applications
```

## Package manager

Install pasting the snippet from [https://sublime.wbond.net/installation](https://sublime.wbond.net/installation) into Sublime Text
